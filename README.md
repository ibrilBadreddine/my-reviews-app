## How to use?

## Settings

### Widget

| Setting | Options / Details                                                        |
| ------- | ------------------------------------------------------------------------ |
| Layout  | Carousel, Floating                                                       |
| Type    | Dynamic (c), Video slider (c), Testimonials (c), Sidebar (f), Pop-up (f) |

---

### Appearance

| Setting                      | Options / Details                      |
| ---------------------------- | -------------------------------------- |
| Color                        | —                                      |
| Intensity                    | —                                      |
| Font                         | —                                      |
| Corner radius                | Flat, Soft, Softer                     |
| Vertical spacing (desktop)   | —                                      |
| Horizontal spacing (desktop) | —                                      |
| Vertical spacing (mobile)    | —                                      |
| Horizontal spacing (mobile)  | —                                      |
| Number of columns on desktop | Available for Dynamic and Video slider |

---

### Reviews

| Setting | Options / Details |
| ------- | ----------------- |
| Author  | Optional          |
| Content | Required          |
| Rating  | Required          |
| Image   | Optional          |
| Video   | Optional          |
| Product | Optional          |
